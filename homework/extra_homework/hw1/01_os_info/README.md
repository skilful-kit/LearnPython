## Информация о системе
### Что нужно сделать
Соберите информацию об операционной системе и версии Python. 
Необходимая информация:
- Название операционной системы
- Версия python
    

### Результат
- Создан файл «os_info.txt» с информацией об операционной системе и версии Python.
- В файле должна быть записана следующая информация:

    OS info is 
    Linux
    Python version is 3.13.11 (main, Dec  6 2025, 08:52:51) [GCC 11.4.0]
